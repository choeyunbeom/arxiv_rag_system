# arXiv RAG System

A Retrieval-Augmented Generation system for academic papers, built with a small language model (Qwen3 4B) to demonstrate how engineering techniques (RAG + QLoRA fine-tuning) can maximise performance of compact models.

## Tech Stack
- **LLM**: Qwen3 4B (via Ollama, Metal GPU accelerated)
- **Embedding**: mxbai-embed-large
- **Vector DB**: ChromaDB
- **API**: FastAPI
- **UI**: Streamlit
- **Fine-tuning**: QLoRA with PEFT
